# my-first-learning-project
这是我的第一个GitHub公共仓库，是我的第一个自主开发的学习项目。
## 主要功能
一个简易记账本程序，帮助你管理个人财务。
## 快速开始
### 第一步：下载代码
1. 点击绿色的 "Code" 按钮
2. 选择 "Download ZIP"
3. 解压到你的电脑
### 第二步：运行记账本
1. 打开解压后的文件夹
2. 双击 `main.py` 文件
## 可能的问题
### Q1.如果双击'main.py'没反应怎么办？
A1.确保您安装了python最新版。具体方法如下：
1.访问https://www.python.org/
2.下载最新版python
3.安装时一定要勾选"Add Python to PATH"
